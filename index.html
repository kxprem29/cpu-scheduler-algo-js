FCFS

First come – First served (FCFS), is the simplest scheduling algorithm. In this algorithm, the process that comes first will be executed first and next process starts only after the previous gets fully executed.

02

SJF

Shortest job first (SJF), is a scheduling policy that selects the waiting process with the smallest execution time to execute next. SJF is a non-preemptive algorithm.

Read More Demo

03

SRTF

Shortest Remaining Time First (SRTF) is the preemptive version of Shortest Job Next (SJN) algorithm, where the processor is allocated to the job closest to completion.

Read More Demo

04

Priority Scheduling

Priority scheduling is a non-preemptive algorithm. The process with the highest priority is to be executed first and so on. Processes with the same priority are executed on a first-come first served basis.
